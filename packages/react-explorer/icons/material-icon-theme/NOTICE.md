# Material Icon Theme — third-party icons

The SVG files in this directory are redistributed from the
[Material Icon Theme](https://github.com/material-extensions/vscode-material-icon-theme)
project by **Material Extensions**, used here under the terms of the MIT
license (see `LICENSE` in this directory).

- Upstream repository: https://github.com/material-extensions/vscode-material-icon-theme
- npm package: https://www.npmjs.com/package/material-icon-theme
- Authors: Philipp Kief and the Material Extensions contributors

No modifications have been made to the SVG sources themselves; they are
served as-is by this extension's webview so the React Explorer can display
file/folder icons consistent with the upstream theme. The file-name → icon
mapping is consumed at runtime from `material-icon-theme/dist/material-icons.json`.
